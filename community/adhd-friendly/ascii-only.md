# ASCII Only

No emojis. No special characters. Plain text unless Owner explicitly requests otherwise.

## Rules

- All AI output must use ASCII characters only.
- No emoji in file names, file contents, commit messages, or chat responses.
- No unicode decorative characters (arrows, bullets, boxes) -- use plain text equivalents.
- This applies to all output boundaries: files, APIs, emails, published content, and chat.

## Why

Emojis and special characters create visual noise that competes for attention. Consistent plain-text output is faster to scan, easier to search, and renders predictably across all tools and terminals.
